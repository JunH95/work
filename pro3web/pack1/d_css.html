** 스타일 시트 **
HTML문서에 다양한 레이아웃과 꾸미기를 위한 표현적인 언어 CSS 사용 (cascade style sheet)
서식 : 선택자{속성 : 값 ; ...}
선택자 종류는  
1) 요소명 : body tag 내의 특정 tag를 직접 지적해서 선택
2) . : class 선택자 - class 속성값, 보유하고 있는 요소를 복수 선택 가능
3) # : id 선택자, 단수 선택 가능(복수X)
스타일은 하위 요소에 상속됨

css 적용방법 : inline, internal, external
